# AI Agent Context for Bootstrap

## Project Overview
This is the **Bootstrap CLI** tool used to scaffold SaaS applications.
- **Stack**: Python, Typer, Copier, Hatchling.
- **Templates**: Located in `src/bootstrap/templates`.

## Development
- Run tests: `task test`
- Run linter: `task lint`
- Sync dependencies: `task setup`

## Key Files
- `src/bootstrap/commands/init.py`: Main logic for project generation.
- `install.sh`: Installer script.
